ソフトウェア演習（Java：Ａコース）

ソフトウェア演習Ａコースの演習課題と、関連する説明を掲載した HTML テキストの導入ページです。

演習内容に関して

  • 演習課題の提出は Moodle （https://moodle2.imc.tut.ac.jp）を用います。 ただし、Moodle を利用する際のブラウザには Firefox を用いて下さい。
  • 課題提出の際には、演習室の環境で正常に実行される事を確認してから提出して下さい（すなわち、演習室の環境で正常に動作する事が条件として課せられます）。
  • 上記のスケジュール表はあくまで予定であり、期日や内容を途中変更する可能性があります。

講義内容に関して

教科書および参考書

本演習では、特定の（市販の紙の）教科書は使用しません。また、演習の科目なので、課題作成にできるだけ多くの時間を割けるように、毎週のトピックに関する口頭での説明以外には、課題に対する補足的説明は必要最小限に抑えます。ただし、要所を理解するために必要となる情報はこのテキストで提供しており、より詳細を調べるのに役立つリンク情報も載せています。さらに、より深い学習に役立つサイトを以下の参考サイトに紹介していますので、自分の実力に適したものを必要に応じて選択して自習に役立てて下さい。
